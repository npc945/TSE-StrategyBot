# 台股模擬自動交易系統

## 專題簡介
本專題模擬台股自動化交易流程，整合技術面策略測試、新聞情緒分析、LineBot 推播與網頁展示功能。

## 功能構想
- 多種技術指標策略回測與勝率統計
- 使用者輸入持股代碼，自動分析並回傳評估報告
- 爬取金融新聞，搭配 AI 模型做情緒判斷
- 模擬買賣紀錄自動發送至 LineBot
- 最後整合成可操作的網站平台

## 使用技術
- Python, Flask, Line Messaging API, BeautifulSoup,FinMind
- MySQL（資料儲存）
- Git + GitHub（版本管理）

## 專案期間
2025 年 5 月 ～ 2026 年 5 月
